# TextilPro - Sistema de Gestión Textil Profesional

## 📋 Descripción
Sistema de gestión integral para taller textil desarrollado con Python, Tkinter y SQLite. 
Implementa arquitectura MVC, validaciones profesionales, exportación de datos y gestión de imágenes.

## 🎯 Propósito del Proyecto
Demostrar la aplicación práctica de conceptos avanzados de programación:
- Programación Orientada a Objetos (POO)
- Patrón arquitectónico Modelo-Vista-Controlador (MVC)
- Interfaces gráficas profesionales con Tkinter
- Integración con bases de datos SQLite
- Procesamiento de imágenes con PILLOW
- Exportación de datos a Excel y PDF

## 🛠️ Tecnologías Utilizadas

| Tecnología | Versión | Propósito |
|------------|---------|-----------|
| Python | 3.11+ | Lenguaje principal |
| Tkinter | - | Interfaz gráfica |
| SQLite | 3.x | Base de datos |
| PILLOW | 10.0+ | Procesamiento de imágenes |
| openpyxl | 3.1+ | Exportación a Excel |
| reportlab | 4.0+ | Exportación a PDF |
| tkcalendar | 1.6+ | Selector de fechas |

## 📦 Requisitos Previos

1. **Python 3.11 o superior** instalado
2. **Pip** (administrador de paquetes de Python)
3. **Git** (opcional, para clonar el repositorio)
